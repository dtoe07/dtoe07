Oregon Liquor Checker Bot

This bot scrapes the Oregon Liquor Search website daily using requests and BeautifulSoup. Generates maps using Folium. It has been running automatically since 2021 to monitor high-demand product availability.

What it does:
  • Fetches daily product status
  • Parses structured HTML data
  • Creates maps with all the stores that has stocks
  • Hosts the maps on GitHub Pages
  • Sends emails about product availability
  • Runs continuously via crontab with Git Action
